### What is Astral?

---

Astral Network wants to connect people, merchants and institutions via its platform offering a permissionless, immutable, infinitely scalable network solution. 

### Key Features

* Astral utilizes a novel [block-lattice](https://github.com/nanocurrency/raiblocks/wiki/Block-lattice) architecture, unlike conventional blockchains used in many other cryptocurrencies.
* The network requires minimal resources, no high-power mining hardware, and can process high transaction throughput.
* Offers instantaneous transactions with zero fees and unlimited scalability, making Astral an ideal solution for peer-to-peer transactions.
